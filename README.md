# Sales-Performance-Analysis
Sales Performance project using Excel,PowerBI,SQL

## Project Overview
This project demonstrates an end-to-end Sales Analysis using Excel, SQL, and Microsoft Power BI. The data was cleaned and prepared in Excel, analyzed using SQL queries, and visualized through an interactive Power BI dashboard to generate meaningful business insights.

## Tools & Technologies
- Microsoft Excel
- SQL
- Microsoft Power BI

## Workflow
1. Data Cleaning using Excel
2. Data Analysis using Excel and SQL
3. Dashboard Development using Power BI
4. Business Insight Generation

## Dashboard Features
- KPI Cards
  - Total Sales
  - Total Orders
  - Total Profit
- Sales by Region
- Monthly Sales Trend
- Category Distribution
- Top 10 Customers
- Interactive Slicers
  - Region
  - Category
  - Order Date

## Key Business Insights
- Identified the best-performing sales regions.
- Analyzed monthly sales trends.
- Compared category-wise sales performance.
- Identified top customers based on sales.
- Enabled interactive filtering for detailed business analysis.
